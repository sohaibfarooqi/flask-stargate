Stargate Docs!¶

Note

This project is currently under development.

Stargate is a framework for exposing RESTFul JSON APIs using Flask and SQLAlchemy.Currently it provides GET, POST, PATCH, DELETE endpoints for resources, pagination, filters, sorting, grouping, resource expansion and partial responses.
